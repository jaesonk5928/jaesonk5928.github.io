CYOA Project

A 10th Grade Project made by Jaeson K and Darren L


Basically, this project is about choosing your own path (choices) to see where you end up in. This helped me being more in context of what Git and Github is about. On yeah, by the way, we used HTML in the files. The adventure I am doing is to try to Prank Mr. Mueller . Me and my partner (Darren) are supposed to make different paths and some are successful while others aren't. I make a couple of choices that would be either fail to prank Mr. Mueller or ways to successfully prank Mr. Mueller . A sneak peek of it would be to either enter the classroom or get the hallpass and leave . A challenge I faced during making this project is that I needed to link all the internal files into other files. But it was hard to link it at that time. I ask people on Slack and got assistance, so I was able to link files together. If i had more time, I would've added images in the files to make it more deciving.
